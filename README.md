# ligand-atom-renamer

A Python tool for automatic atom renaming in ligand PDB files. Converts generic atom names (e.g., C, N, O, H) into unique atom identifiers (e.g., C1, C2, N3, H1, H2) while preserving the original molecular coordinates, connectivity records, and PDB formatting.

## 1. Features

* Automatically processes all HETATM records in a ligand PDB file
* Generates unique atom names based on atom type and serial number
* Sequentially renames hydrogen atoms (H1, H2, H3, ...)
* Preserves original coordinates and molecular geometry
* Retains CONECT records, TER records, and metadata
* Maintains proper PDB column formatting
* Produces ligand files compatible with docking and parameterization workflows

## 4. Usage

Run the script by providing a ligand PDB file as input:

```bash
python rename_ligand_atoms.py ligand.pdb
```

### Output

The script generates a new PDB file:

```text
ligand_renamed.pdb
```

### Output Characteristics

* Unique atom names
* Original coordinates preserved
* Original bonding information preserved
* Proper PDB formatting retained
* Compatible with docking and molecular modeling software


---

## 6. Important Notes

* Only HETATM records are modified
* REMARK, TER, CONECT, and END records are preserved unchanged
* Heavy atoms are renamed using their element symbol and atom serial number
* Hydrogen atoms are renamed sequentially as H1, H2, H3, ...
* Molecular coordinates and connectivity are not altered
* Intended for technical preprocessing rather than structural modification

---

## 7. Use Cases

* Preparing ligands for molecular docking workflows
* Generating unique atom names required by docking software
* Standardizing ligand structures before parameterization
* Preprocessing ligands for HADDOCK, AutoDock, AutoDock Vina, Glide, GOLD, and related tools
* Cleaning ligand files generated by molecular editors and structure builders
* Improving compatibility with topology and force-field generation pipelines
